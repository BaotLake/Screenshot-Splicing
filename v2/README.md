
# splicing V2

由视频取帧拼接屏幕长截图

## 用法
```shell
./splicing.py ./img/a.mov
```

# 安装依赖
- numpy 矩阵运算
- Pillow 图片处理
- openCV 视频取帧

```shell
# numpy
pip install numpy

# pillow
pip install pillow

# openCV 
conda install opencv
```

opencv依赖众多，pip安装后import导入可能会报错，推荐使用conda安装opencv
